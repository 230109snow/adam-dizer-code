The Application

The Application is a web platform to host your professional profile.
The Application is built for easy use to add new projects to be displayed for your profile.
The Application has multiple sections outlined below

The Navigation Bar

The Navigation Bar allows seamless transitions between different components within the profile.

About Me

The About Me compontent is an simple profile to display important professional information about you!

Helpful Links

The Helpful Links is a component that hosts a list of web addresses for professional reference.
The Helpful Links loads the external web links within the component for easy searching and viewing transition.

My Projects

The My Projects component hosts projects to be showcased for professional portfolios.
The My Pojects component is designed for easy implimentation of new projects build within the angular framework.
